Answers

1. What is the default value of the position property in CSS?

C. Static


2. Which CSS positioning type allows an element to be positioned relative to its normal position?

B. Relative

3. Which CSS positioning type removes an element from the normal document flow with no space created for the element in the page layout?

C. Absolute


4. Which CSS positioning type positions an element relative to the browser window?

D. Fixed


5. Which CSS positioning type is a hybrid of relative and fixed positioning?

D. Sticky


6. Which CSS property would you use to move a relatively positioned element 20px to the right of its normal position?

D. right


7. If two elements are absolutely positioned, which CSS property controls which one appears on top?

A. z-index


8. Which positioning type is not affected by scrolling?

C. Fixed


9. Which of the following is not a valid value for the position property?

D. Center


10. Which positioning type would you use to create a sticky header that remains at the top of the viewport when scrolling?

D. Sticky
